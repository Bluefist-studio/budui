Budget UI Tutorial & Help

Welcome to the Budget UI help page! This guide will walk you through the main features and functionalities of the Budget UI application, designed to help you manage your biweekly budget effectively.

Math & Biweekly Budget Cycle

  • Biweekly Budget: All calculations are based on a two-week period, matching most payroll schedules.
  • Totals: For each category, the app adds the "First" and "Second" amount fields. The Total field is their sum once marked as paid.
  • Forecasts & Balance: Income Forecast and Expense Forecast show expected totals for the period. Account Balance is Income Forecast - Expense Forecastand should match what is currently in your account.
  • Current Expenses: Shows the sum of all paid expenses for the current period.
  • Progress Bar: Visually shows remaining budget vs. spending.

Income Section

  • Expand/collapse the section with the accordion button.
  • Lock/unlock fields to prevent accidental edits and allow marking income as received.
  • Edit category names and enter amounts for each income source.
  • Totals update automatically.

Expense Section

  • Expand/collapse expense categories (e.g., Housing, Transport).
  • Lock/unlock fields for each category to prevent accidental edits and allow marking expenses as paid.
  • Edit category names and enter amounts for each expense type.
  • Totals update automatically.

Sub-Category Placeholders

  • Some expense categories include empty fields labeled "Sub-category." Type a name and amount to make them visible and included in calculations.
  • Clear the name and amount to hide unused sub-categories and keep your budget tidy.

How to Edit Your Budget

  1. Expand the section you want to edit.
  2. Unlock fields.
  3. Edit category names or enter new amounts.
  4. Totals update automatically. Review the summary bar for changes.
  5. Lock fields when finished.

Paid Mechanic

  • Mark expenses as paid for the current biweekly period to track which bills have been settled.
  • While the section is locked, click an expense entry to mark it as paid. Paid expenses are visually highlighted and included in the Current Expenses total.
  • This helps you avoid double-paying and gives a clear view of what still needs to be paid.
  • Monthly Payments: For bills that are paid only once a month (e.g., rent, insurance), the app tracks their payment status within the biweekly cycle. If a monthly payment is marked as paid in one biweekly period, both biweekly periods are updated accordingly. This ensures you only pay these bills once per month and prevents confusion.
  • In the profile menu, you can find the "Reset All Paid Status" button: Use this button to clear the paid status for all expenses and income entries in the current cycle. This is useful at the start of a new month, allowing you to begin tracking payments from scratch. Be careful: resetting will mark all entries as unpaid, so only use it when you want to start fresh.
